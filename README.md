# 📚 Book Data Scraper

A Python web scraper that extracts book data from [books.toscrape.com](https://books.toscrape.com) and saves it to a clean Excel file.

## 📊 What It Scrapes

| Column | Description |
|---|---|
| Title | Full book title |
| Price | Book price in £ |
| Rating | Star rating (1–5) |
| Availability | In stock / Out of stock |
